#一、项目简介
***
本项目是vue.js(2.0)基于webpack1.0的项目脚手架，它可以让你快速构建vue项目。

#二、项目安装
***
##1、安装所需要的包
`npm install`

##2、运行指令
`npm run dev`

##3、预览效果
输入网址：<br>
   http://localhost:8188/
   
效果如下：<br>
   ![Image text](https://github.com/duanliang920/vue-webpack-project/blob/master/dev.jpg)

##4、编译打包
`npm run build`
   

温馨提示：如果没有安装node和webpack的朋友，请先安装。
